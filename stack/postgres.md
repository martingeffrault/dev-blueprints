## Philosophy (2025)

PostgreSQL 16+ emphasizes JSON capabilities, parallel query improvements, and logical replication.

**Key shifts:**
- JSONB is mature — use it for semi-structured data
- Full-text search is production-ready
- Prefer CTEs for readability (optimized since PG12)
- Use `GENERATED ALWAYS AS` for computed columns
- Logical replication for zero-downtime migrations
- Connection pooling is essential (PgBouncer, Supavisor)
