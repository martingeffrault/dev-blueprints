## Philosophy (2025)

Node.js 22+ focuses on modern JavaScript support and native TypeScript execution.

**Key shifts:**
- Native TypeScript execution with `--experimental-strip-types`
- ES Modules are the default — CommonJS is legacy
- Built-in test runner (`node --test`)
- Built-in watch mode (`node --watch`)
- Fetch API is stable and default
- WebStreams API available
- Permission model for security
