## Philosophy (2025)

Playwright is the standard for end-to-end testing. Cross-browser, reliable, fast.

**Key shifts:**
- `test.step()` for readable tests
- Component testing for isolation
- Visual comparisons for UI testing
- Parallel execution by default
- Trace viewer for debugging
- API testing support
