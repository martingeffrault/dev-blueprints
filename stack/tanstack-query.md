## Philosophy (2025)

TanStack Query 5 simplifies the API and improves SSR support.

**Key shifts:**
- Suspense is first-class (`useSuspenseQuery`)
- Streaming SSR support
- Simplified optimistic updates
- Query invalidation is still the core mental model
- `useQuery` for reads, `useMutation` for writes
- Works great with Server Components
