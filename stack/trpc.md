## Philosophy (2025)

tRPC 11 improves streaming, React Query integration, and edge support.

**Key shifts:**
- End-to-end type safety without code generation
- Procedures = type-safe endpoints
- React Query integration is first-class
- Server-side calls in Server Components
- Streaming responses with async generators
- Input validation with Zod
