## Philosophy (2025)

React in 2025 embraces Server Components as the default mental model. The framework has shifted from client-first to server-first thinking, with RSC (React Server Components) being the primary way to build applications.

**Key shifts:**
- Server Components by default, Client Components opt-in
- No more useEffect for data fetching — use async Server Components
- Suspense for loading states is now the standard
- Actions for mutations (form handling)
- CSS-in-JS libraries discouraged in favor of CSS Modules, Tailwind, or vanilla CSS
- The React Compiler (React Forget) handles memoization automatically
