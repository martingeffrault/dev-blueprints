## Philosophy (2025)

Vitest is THE testing framework for Vite projects. Jest compatibility with modern DX.

**Key shifts:**
- Jest-compatible API
- First-class TypeScript support
- Vite's transform pipeline = fast
- Browser mode for real browser testing
- In-source testing with `if (import.meta.vitest)`
- Workspace support for monorepos
