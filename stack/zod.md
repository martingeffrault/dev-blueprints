## Philosophy (2025)

Zod is the standard for runtime validation with TypeScript inference.

**Key shifts:**
- Schema = runtime validation + TypeScript type
- Use at API boundaries (forms, API responses)
- `z.infer<>` for type extraction
- Composable schemas (merge, extend, pick, omit)
- Works great with React Hook Form, tRPC, Hono
