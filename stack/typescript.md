## Philosophy (2025)

TypeScript 5.x focuses on performance, type inference improvements, and stricter defaults.

**Key shifts:**
- Use `strict: true` — no exceptions
- Prefer `satisfies` over type assertions
- Template literal types for string patterns
- Discriminated unions over optional properties
- `const` type parameters for literal inference
- `using` declarations for resource management (5.2+)
- No `any` — use `unknown` and narrow
