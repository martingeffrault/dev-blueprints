## Philosophy (2025)

Tailwind 4 is a complete rewrite using Rust (Oxide engine), with CSS-first configuration.

**Key shifts:**
- Configuration in CSS, not JS (`@theme` directive)
- Native CSS cascade layers
- Container queries built-in
- No more `tailwind.config.js` for simple projects
- Automatic content detection
- First-party Vite plugin
- New color palette system
