## Philosophy (2025)

Prisma 6 focuses on edge compatibility, performance, and type safety improvements.

**Key shifts:**
- Prisma Accelerate for connection pooling at edge
- Query extensions for reusable logic
- Client extensions for custom methods
- Improved raw SQL support with `$queryRawTyped`
- Prisma Pulse for real-time data sync
- Better serverless/edge support
