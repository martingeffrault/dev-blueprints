## Philosophy (2025)

Vite 6 focuses on environment API and build performance.

**Key shifts:**
- Environment API for SSR/client separation
- Rolldown integration (Rust bundler) coming
- `vite.config.ts` is the standard
- First-class Tailwind v4 support
- Use `vite-plugin-*` ecosystem
- React plugin for Fast Refresh
