## Philosophy (2025)

Drizzle is SQL-first — if you know SQL, you know Drizzle. Zero abstraction over the database.

**Key shifts:**
- SQL-like syntax, not abstraction
- Schema in TypeScript = type safety
- Zero dependencies, tiny bundle
- Perfect for serverless/edge
- No query builder magic — what you write is what runs
- Drizzle Kit for migrations
